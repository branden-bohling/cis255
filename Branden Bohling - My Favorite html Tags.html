My favorite HTML tags

Introduction. The purpose of this essay is to describe my three favorite Hypertext Markup Language (HTML) tags. I prepared this HTML document as a sample for students learning HTML.

<a>. The anchor tag is my #1 favorite HTML tag because it most fundamentally embodies the purpose of the World Wide Web: to enable documents to be linked to other documents. According to w3.org, "An anchor is a piece of text which marks the beginning and/or the end of a hypertext link." So without anchor tags, links would not exist. And without links the Web would not exist. That's why anchor tags seem pretty darn important.

<hr>. The horizontal rule tag is often overlooked, but it can be quite handy for organizing sections of a web page.

<svg>. Many people are unaware of the Scalable Vector Graphics (SVG) tag. SVG defines vector-based graphics in eXtensible Markup Language (XML) format.

Conclusion. We may debate which tags are really the best, but in my opinion, the anchor tag, the horizontal rule tag and the SVG tag stand out as particularly useful and/or interesting. For the moment at least, they are my favorites.
