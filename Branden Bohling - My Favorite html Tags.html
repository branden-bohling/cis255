My favorite HTML tags

Purpose of this page This html page was made to demonstrate a few of my favorite HTML tags and their purpose, these may not be the most useful tags, but I find them interesting and fun to use in their own right
  1. <ol or ul> The list tag in HTML is one which I will be using for the rest of this page in order to list my favorite html tags. The <ol> tag defines an ordered list 1,2,3,... (the one I am currently using) while the <ul> tag defines a list which is simply bullet points. This is very useful for informitive websites such as this one.   2. <fieldset> Another tag which I find fun and useful for organizing web pages is the <fieldset> tag. This tag allows for a map-like legend to show a certain patch of text. I have used the fieldset tag down below in the Course Lookup section as I thought It would make the section stand out a bit more. The fieldset tag can be used with the legend tag to label the field
  3. <span> The <span> tag allows for text to appear when a certain element is scrolled over. This is particuarly useful for adding descriptions to pictures or extra information to blocks of text. Scroll over this text to see the message behind it.
